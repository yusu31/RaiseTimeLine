# RaiseTimeLine

X（旧Twitter）のタイムライン形式を参考にした、テキストベースのSNS風Webアプリです。
プログラミングスクール RaiseTech の課題として、学習目的で開発しています。

> アプリ名「RaiseTimeLine」は仮称です。

## 主な機能（予定）

- ログイン機能（サインアップ / ログイン / ログアウト）
- タイムライン機能（投稿の作成・時系列表示）
- コメント機能（コメント数の表示つき）
- いいね機能（いいね数の表示つき）
- 画像投稿機能

### Xとの違い（差別化ポイント）

- インプレッション数（表示回数）は表示しない
- リツイート（リポスト）機能は作らない

## 技術スタック（予定）

| 役割 | 技術 |
|------|------|
| フロントエンド | React + TypeScript |
| バックエンド | Java + Spring Boot |
| データベース | PostgreSQL |
